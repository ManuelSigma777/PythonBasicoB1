# PythonBasicoB1

# 1. ¿Qué es una Base De Datos Relacional?

# Es un tipo de base de datos en el cual la información se acomoda en tablas y está conectada entre sí con algo llamado “Keys”.

# 2. ¿Qué es una Tabla en el contexto Base De Datos Relacionales?

# Es un conjunto de filas y columnas al estilo de Excel que almacena temas o usuarios.

# 3. ¿Qué es una relación en el contexto de Base De Datos Relacionales?

# Es la relación entre 2 tablas que se hace con claves, primaria y foranea.

# 4. ¿Qué representa una columna en el contexto de Base De Datos Relacionales?

# Una columna  representa una característica  del dato. Ej: En una tabla de alumnos las columnas pueden tener nombre, grupo, etc.

# 5. ¿Qué representa un renglón en el contexto de Base De Datos Relacionales?

# Representa un dato completo. Ej: Alumnos : Nombre - Grupo-Años.

# 6. ¿Qué es un motor de base de datos relacional?

# Es un software en el cual se puede consultar, crear y administrar informacion de una base de datos relacional con SQL.

# 7. ¿Cuáles son los motores de bases de datos relacionales mas populares?

# MySQL.
# PostgreSQL.
# Microsoft SQL Server.
# Oracle Database.
# MariaDB.

# 8. ¿Cuál es el motor de base datos ligero que no necesita un servidor?
# SQLite, porque guarda toda la base de datos en un solo archivo y no ocupa servidor.

# 9. ¿Qué significa SQL en el contexto de base datos relacionales?
# SQL significa Structured Query Language y es el lenguaje usado para crear, consultar, modificar y administrar bases de datos relacionales.

# 10. ¿Cuáles son los elementos de un sistema de información basado en base de datos relacionales?
# Base de datos: donde se almacenan los datos organizados.
# Motor de base de datos : el programa que administra la base de datos.
# SQL: lenguaje para comunicarse con la base de datos.
# Usuarios: personas que usan o administran el sistema.
# Aplicación o sistema: programa que usa la base de datos
# Hardware: equipo físico donde se ejecuta el motor y se guardan los datos.
# Red: permite que varios usuarios accedan a la base de datos.
